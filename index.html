welcome.js

This is the demo page for welcome.js, a tiny javascript file you can add to any web page to help novice users get the most out of the web.

Do it

To see the script in action, open the console.

On Mac OSX, you can do the following. Sorry I don't know about other systems, but you can probably work it out.

  • Firefox: Tools > Web Developer > Web Console (Opt-Cmd-K)
  • Chrome: View > Developer > Javascript Console (Opt-Cmd-J)
  • Safari: Develop (Must be enabled in options) > Show Web Inspector (Opt-Cmd-I) / Select Console
